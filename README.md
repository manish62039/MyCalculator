
# Calculator App

Multi theme calculator app.


## Features

- Sooth UI
- Change UI in just one click
- Calculations are also exact 


## Tech Stack

**Architecture Pattern:** MVVM

**Libraries Used:** Shared Preferences, data binding, Object hunter(Calculation helper)
